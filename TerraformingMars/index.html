Terraforming Mars

Description

Terraforming Mars is a 1-5 player board game in which the players compete to gain victory points awarded for contributions to terraforming the atmosphere and advancing human infrastructure on Mars.

Publishing Info

  • Designer: Jacob Fryxelius
  • Artist: Isaac Fryxelius
  • Publisher: FryxGames
  • Year Published: 2016

Game Categories

  • Genre: Eurogame
  • Core Mechanic: Card Drafting
  • Themes: Science Fiction, Space Exploration
  • Game Length: 1.5 - 2 Hours
